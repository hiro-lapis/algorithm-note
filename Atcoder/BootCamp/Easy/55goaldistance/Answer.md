- ある座標(y1,y2,yD)と、ある座標(z1,z2,zD) の間の距離は `(yi - zi)^2` で求められる



```

2次元に3つの点がある

x1~x2 間の距離: ((1 - 5) + (2-5))^2 => 5
x2~x3 間の距離: ((5 - -2) + (5 -8))^2 => √58
x3~x1 間の距離: ((-2 - 1) + (8 - 2))^2 => 3√5

よって、整数になるのは1 つだけ
```
