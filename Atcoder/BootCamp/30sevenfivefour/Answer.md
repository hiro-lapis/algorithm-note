## 問題文の把握

- 入力値 S が与えられる
- 文字は1,2,...9 までの文字である
- 753 に近い文字ほど良い
- 与えられた文字列 S のうち 3 文字を切り出し、入力値の中で最も753 に近い値を求めよ

- 制約
- 4 <= S <= 10
- 1,2,3...9

```
case1
1234567876
=> 34

case2
35753
=> 0

case3
1111111111
=> 642
```
`
## ロジック

「753 に近い値を探す」となった場合、百の位が 7 に近い値から探すのが第一になる  
百の位の値さえ定まれば、それ以降の文字の痴漢は NG なので自然と近似値が定まる  
以上を踏まえて、7 に近い値が 1 ~ 8 文字目にあるかどうかを探索する
