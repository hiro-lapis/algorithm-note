#bit演算

## 問題文の把握

- 正の整数 N が与えられる。各桁の値は 0 以外である
- N の桁数を k とする。N の桁を 0~k-1 まで決して、残った桁をそのままの順序で結合することで 3 の倍数を作りたい　　
- 3　の倍数を作成できるか判定し、できるなら消す桁が最小の数を求めよ
- 制約
- 1 <= N <= 10^18

```
N

case1
35
=> 1 (3)

case2
369
=> 0 (369)

case3
6227384
=> 1 (6227384)

case4
11
=> -1
```
